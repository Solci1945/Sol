Guía: Subir un archivo a GitHub usando Git

Paso 1: Instalar Git

Si aún no tienes Git instalado, puedes descargarlo e instalarlo desde aquí. Sigue las instrucciones de instalación para tu sistema operativo.

Paso 2: Configurar Git

Abre una terminal y configura tu nombre de usuario y correo electrónico:

git config --global user.name "Tu Nombre"
git config --global user.email "tuemail@example.com"

Paso 3: Crear un repositorio en GitHub

Ve a GitHub y crea un nuevo repositorio. Anota la URL del repositorio, la necesitarás más adelante.

Paso 4: Inicializar un repositorio local

En la terminal, navega a la carpeta donde deseas crear tu proyecto y ejecuta:

git init

Paso 5: Añadir un archivo

Crea un archivo nuevo en tu proyecto (por ejemplo, archivo.txt). Luego, agrega este archivo al área de preparación:

git add archivo.txt

Paso 6: Hacer un commit

Realiza un commit con un mensaje descriptivo:

git commit -m "Añadir archivo.txt"

Paso 7: Enlazar el repositorio local con GitHub

Enlaza tu repositorio local con el repositorio remoto en GitHub usando la URL que anotaste en el paso 3:

git remote add origin https://github.com/tuusuario/turepositorio.git

Paso 8: Subir los cambios a GitHub

Sube tus cambios al repositorio remoto en GitHub:

git push -u origin master

Si es la primera vez que subes, Git te pedirá que ingreses tus credenciales de GitHub.

¡Listo!

Ahora tu archivo debería estar en tu repositorio de GitHub. Puedes verificarlo accediendo a la URL del repositorio en tu navegador.
